# ASHBORN — Contexto do Projeto

## O Jogo
Action RPG 2D top-down Dark Fantasy em Godot 4.4 com GDScript.
Protagonista: Kael, ferreiro que absorve essências de inimigos derrotados.
Referências: Diablo, Mu Online, Hades, Dead Cells.

## Regras de Desenvolvimento
- Um sistema por vez — nunca avançar sem o anterior testado
- Código sempre comentado (uma linha por função)
- Explicar o que foi feito em português simples após cada entrega
- Placeholders são válidos até a Fase 5
- Instruir commit no GitHub após cada sistema funcionando

## Stack
- Engine: Godot 4.4 / GDScript / GUT / Git + GitHub
