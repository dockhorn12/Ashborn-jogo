# Ashborn

> "Todo lendário um dia foi ninguém."

Action RPG 2D top-down Dark Fantasy desenvolvido em Godot 4.4.
Você é Kael, um ferreiro que absorve as essências dos inimigos que derrota — ficando mais forte a cada combate.

Referências: Diablo, Mu Online, Hades, Dead Cells.

## Stack

- Engine: Godot 4.4
- Linguagem: GDScript
- Testes: GUT
- Versionamento: Git + GitHub

## Como rodar

1. Instale o [Godot 4.4](https://godotengine.org/)
2. Clone este repositório
3. Abra o Godot, clique em **Import** e selecione o arquivo `project.godot`
4. Pressione **F5** para rodar
